Student Grade Database

Data Structuring with Python

In this project, I will practice my Python writing skills and also my data structuring skills. I will create a sample for Student grade database with Python.


Relational Database Structures

I took two approaches to create two different databases. Both databases are relational databases. "A relational database is a type of database that stores and provides access to data points that are related to one another. Relational databases are based on the relational model, an intuitive, straightforward way of representing data in tables."(Oracle, ) The relational database is a model that includes basic data structures: the data tables, views, and indexes. The administrators can manage data storage without affecting access to that data as a logical structure.

As indicated above, there are two tables listed that we need to input. One includes the student names(Name), problem set ID (Pset), and the grade they score on this problem (Grade). Another table includes problem set ID(PsetID), and the maximum score one can score on this problem(MaxScore.) Some potential questions we can ask with this data we have are:

What is Fred's total for all the Psets?
What is Fred's average percentage for all the Psets?
What is the average score for Pset 1?
What is the average weighted percentage score for all the Psets?

Another approach is to create more tables. Here I created three tables: Student ID, Problem set ID, and Grades. With both methods we need to require We can match them and created a database accordingly. We can use primary key and foreign key to connect them. A primary key is used to ensure data in the specific column is unique, and a foreign key is a column or group of columns in a relational database table that provides a link between data in two tables. According to the first approach with only two tables, we can set the StudentID as the primary key and the PsetID as the foreign key in table 1, and PsetID as the primary key in table 2. By using the PsetID key, we can link two tables together. According to the second approach with three tables, we can set StudentID as the primary key for table 1, PsetID as the primary key for table 2, and Grade as the primary key for table 3 with StudentID and PsetID both as foreign keys. This is how we connect three tables together. We implemented these two approaches on python using list, dictionaries, and functions. We mainly used add functions to input data; get functions to return data with functions.
